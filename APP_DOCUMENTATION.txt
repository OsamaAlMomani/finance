1. APP OVERVIEW & FEATURES
================================================================================

The Advanced Finance App is a comprehensive desktop and web-based personal 
finance management system that helps users track income, expenses, budgets, 
goals, and make informed financial decisions with powerful analytics and 
forecasting tools.

CORE FEATURES:
==============

✓ Transaction Management
  - Track income and expense transactions
  - Categorize spending (Housing, Food, Utilities, Entertainment, etc.)
  - Filter by date, category, type
  - Edit and delete transactions
  - Recurring transaction support

✓ Budget Management  
  - Create monthly budgets for categories
  - Set spending limits
  - Track budget progress with visual indicators
  - Compare actual vs budgeted spending
  - Category-based budget allocation

✓ Goal Tracking
  - Create savings goals with target amounts
  - Track progress with visual progress bars
  - Set deadlines and monthly targets
  - Contribution tracking for each goal
  - Goal completion status

✓ Bill & Reminder Management
  - Track recurring bills
  - Set payment reminders (days before due)
  - Mark bills as paid/unpaid
  - Auto-pay status tracking
  - Due date management

✓ Financial Analytics & Dashboards
  - Net worth tracking and visualization
  - Expense breakdown by category
  - Income distribution analysis
  - Cash flow timeline
  - Savings progress charts
  - Burn rate and runway calculations

✓ Advanced Tools
  - Forecast vs Actual: Compare spending to forecasts
  - Cost of Living: Track inflation impact
  - Scenario Planner: Simulate financial futures (A/B/C scenarios)
  - Danger & Safety Meter: Financial risk assessment
  - Calendar Planning: Visualize money events
  - Task Manager: Track financial tasks

✓ Data Management
  - Export to JSON/CSV formats
  - Import from JSON files
  - Full backup and restore functionality
  - Automatic backup scheduling

✓ Monthly Reports
  - Comprehensive financial summaries
  - Income/expense breakdowns
  - Savings rate calculations
  - Month-over-month comparisons
  - Printable reports
  - PDF export capability
  - Insights and recommendations

✓ UI/UX Features
  - Light/Dark theme support
  - Keyboard shortcuts (Ctrl+K for command palette)
  - Toast notifications
  - Loading skeletons
  - Responsive design
  - Smooth animations
  - Search functionality

FREQUENTLY ASKED QUESTIONS
================================================================================

Q: How do I add a transaction?
A: Click the "Quick Add" button or press Ctrl+N. Select income/expense, enter
   amount, category, and confirm.

Q: Can I edit or delete transactions?
A: Yes, go to Transactions page. Each transaction has edit/delete buttons.

Q: How do I set a budget?
A: Navigate to Budgets page and click "Create Budget". Set category limits.

Q: What happens when I exceed my budget?
A: The budget progress bar turns red and you receive a notification warning.

Q: How do I track multiple savings goals?
A: Use Goal Tracker in Tools. Each goal can have its own icon, color, and deadline.

Q: Can I backup my data?
A: Yes, navigate to Data Management > Backup tab. Click "Create Full Backup".

Q: Can I export my data?
A: Yes, you can export to JSON or CSV format in Data Management > Export tab.

Q: What if I lose my data?
A: If you have a backup, use Data Management > Restore to recover your data.

Q: Can I use the app offline?
A: Yes, all data is stored locally (localStorage and SQLite).

Q: Does the app sync across devices?
A: Currently no automatic sync. You can manually export/import to transfer data.

Q: Can I share my financial data?
A: You can export and share files, or share individual reports via email.

Q: What's the difference between light and dark theme?
A: Light theme for daytime, dark theme for nighttime to reduce eye strain.

Q: Are there mobile versions?

================================================================================
TROUBLESHOOTING
================================================================================

Issue: App won't start
Solution:
1. Clear browser cache/app cache
2. Delete node_modules and reinstall: npm install
3. Check Node.js version (should be 16+)
4. Try: npm run dev to see error logs

Issue: Transactions not showing up
Solution:
1. Check if they're in correct date range
2. Verify category wasn't changed
3. Check browser localStorage isn't full
4. Try: localStorage.clear() in console to reset

Issue: Charts not displaying
Solution:
1. Ensure data exists for charts
2. Check browser compatibility
3. Try refreshing page
4. Check console for JavaScript errors

Issue: Theme not saving
Solution:
1. Check if localStorage is enabled
2. Check browser privacy settings
3. Try switching theme again
4. Clear browser cache and try again

Issue: Export/Import failing
Solution:
1. Verify file format is correct
2. Check file isn't corrupted
3. Try smaller export first
4. Ensure no special characters in file path

Issue: Performance is slow
Solution:
1. Check if you have many transactions (1000+)
2. Try filtering data by date
3. Close other browser tabs
4. Restart browser
5. Check system resources

Issue: Keyboard shortcuts not working
Solution:
1. Check if app window is focused
2. Verify keyboard isn't set to different layout
3. Check browser doesn't override shortcuts
4. Check if extension conflicts
5. Try: Press Shift+? to verify shortcuts are working
